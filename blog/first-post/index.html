Hello, World!! 第一篇文章

2020-08-01 · By Chen,Jian-Fong · 生活紀錄

大家好，我叫 Rex，一位出生於台灣南端的普通宅男。

我畢業於國立宜蘭大學生物機電工程學系 ( 前身為農業機械工程學 )。生機系是強調跨領域整合的學系，在這樣的學習環境的薰陶下，我們有不排斥對於不同領域的接觸的勇氣 ( 大概吧 … )。

在求學過程中為消磨課餘時間順便精進自己，因而選擇自學寫程式 。只要有一台電腦就可以開始學習並且可以立即看到初步成果，相比本科系 ( 機械領域 ) 多數時候學習很難搭配實作驗證學習成果，我因此意外的被資訊領域的魅力所折服。目前正在參與職訓課程，希望能成功轉換跑道成為軟體工程師。

建立這個 BLOG 是希望用來記錄自己的學習歷程或是展示一些小作品，希望透過這樣的方式能協助自己成長為新中理想中的模樣，並成功成為軟體工程師。
